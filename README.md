# `prettier-config`

Prettier config

- Длина строк: 100 символов
- Для отступов используются пробелы
- Одинарные скобки
- Точки с запятой

## Usage

```bash
npm i -D prettier @beckmad/prettier-config
```

Reference prettier-config in your `package.json`:
```json
{
  "name": "my-cool-library",
  "version": "9000.0.1",
  "prettier": "@beckmad/prettier-config"
}
```
